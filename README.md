## Installation
***
A little intro about the installation.
```
$ git clone 
$ cd ../path/to/the/file
$ python -m venv env  ==== to create venv

$ .\venv\Scripts\Activate ==== to active env

$ flask run
```
Side information: The application is only tested in a ```window``` specific  environment
